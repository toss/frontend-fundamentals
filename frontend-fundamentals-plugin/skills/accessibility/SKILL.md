# Web Accessibility Principles

Apply these principles for accessible frontend code:

1. **Proper Structure** - Don't nest interactive elements (no button inside button)
2. **Semantic Meaning** - Label all interactive elements properly
3. **Predictable Interaction** - Use semantic HTML elements (`<button>`, `<a>`)
4. **Don't Rely on Visuals Only** - Provide alt text for images
