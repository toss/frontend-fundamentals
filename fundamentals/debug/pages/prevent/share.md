# 팀과 공유하여 집단지성화 하기
자주 나오는 실수는 혼자만 알지 말고 **팀과 공유해서 다 같이 배워요.** 린터 규칙, PR 체크리스트, 회고 등을 통해 확산시켜요.

## 린터 규칙 추가 

### 예시 1
- 문제: 맞춤법 오류가 빈번히 발생
- 해결방법: ESLint로 금지 단어 설정

```
아니예요 => 아니에요
명사 + 에요 => 이에요/예요
```

### 예시 2
- 문제: “김토스님” 같은 더비 텍스트가 코드에 남는 문제.
- 해결방법: ESLint로 금지 단어 김토스 추가

```
‘김토스’ 포함 시 린트 에러 발생
```
